## 🚀 部署到 GitHub Pages

### 一次性配置

```bash
cd townhall-v06-deploy
git init
git add .
git commit -m "feat: townhall v0.6 投屏稿定稿"
git branch -M main
git remote add origin git@github.com:<你的GitHub账号>/townhall-v06.git
git push -u origin main
```

然后到 GitHub 仓库 → Settings → Pages → Source 选 `main / root` → Save。

约 1 分钟后即可访问：
```
https://<你的GitHub账号>.github.io/townhall-v06/
```

### 后续更新

```bash
git add .
git commit -m "fix: 调整最后一页二维码位置"
git push
```
