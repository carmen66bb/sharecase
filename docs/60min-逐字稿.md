#### 【口播】

【动作 · 停 2 秒 · 给出对策】

>
> "市场调研 Agent / 数据分析 Agent / 报告写作 Agent"——**这种是低维**。
>
> 我按"**思维视角**"建 Agent——

> - 一个站"客户老板视角想生意"——它会说：客户内部也焦虑，比起打竞品更需要把自己漏斗补好

> 三个 Agent 在哪里分歧——**哪里就是真正要重点想的问题**。
>
> 我不看它们的结论——**我看它们吵架的地方**。
>
> 因为只有"真原因"被拨开，三个不同视角的判断才会**同时变化**。
>
> 它们吵架的地方——**就是我以前一个人脑子里看不到的因果链断点**。

【动作 · 举例 · 节奏加快】
